WORKING WITH MULTIPLE APK FILES (batch.sh)
./batch.sh <apkdir> [ remove | install | uninstall | manual | delete | login | signup ]

WORKING WITH 1 SINGLE APK FILE (single.sh)
./single.sh <apkpath> [ remove | install | uninstall | manual | delete | login | signup | launch | download | search ] 

WORKING WITH PLAYSTORE ACCOUNTS AND APP ACCOUNTS (lac_batch.sh)
./lac_batch.sh [ <gmail> <gpassword> minus | <./minus/laXX> install | <apklist> <app-email> <app-password> clean | pull | uninstall | <./minus/laXX> remove ]

WORKING WITH APK PACKAGES EITHER ON THE PHONE OR ON PLAYSTORE (pm.sh)
./pm.sh [ uninstall | list | pull | clean | count | <apklist> search | <package-name> pull ]

WORKING WITH STATISTICS ON LOG FILES OF APKS (stats.sh)
./stats.sh <logdir>

WORKING WITH STATISTICS ON HTML FILES (stats_web.sh)
./stats_web.sh <webdatadir>
